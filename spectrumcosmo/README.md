# SpectrumCosmo

## Setup
1. Connect Neon database in Vercel Storage tab
2. Run neon-schema.sql tables one by one in Neon Query editor
3. Add environment variables in Vercel:
   - POSTGRES_URL (auto-added by Neon)
   - JWT_SECRET (any 32+ char string)
   - ADMIN_USERNAME=admin
   - ADMIN_PASSWORD=SpectrumAdmin2024!
4. Deploy
